# 📘 LeetCode Solutions – DSA Journey

## 👋 Overview
This repository documents my **Data Structures & Algorithms (DSA)** learning journey through LeetCode problems.  
The focus here is **deep understanding**, not rote memorization.

**Each solution reflects:**
- 🧩 Clear problem understanding
- ⚙️ Correct use of patterns (Stack, BFS/DFS, Sliding Window, Binary Search, Recursion, etc.)
- ⚡ Optimized time and space complexity
- 🧠 Learnings that transfer to real-world engineering and ML systems

## 🧠 Learning Philosophy
* **Concepts > Speed:** I prioritize understanding the *why* over just getting the green checkmark.
* **Root Cause Analysis:** I identify why naive solutions fail.
* **First Principles:** I focus on invariants, patterns, and trade-offs.
* **Repetition:** I revisit problems until the logic feels natural.

> *"Confusion is part of learning — clarity is the goal."*
